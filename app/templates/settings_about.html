About

Literapy originated as a final project of the Full Stack Web Development course at the School of Information at UC Berkeley. The course teaches students skills in data modeling, database and API design, responsive front-end design, version control, and deployment using Python, JavaScript, and full-stack frameworks such as Flask. It culminates in a final project in which the students have to apply the learned concepts by implementing a production ready web applications.

The entire source code for Literapy can be found on Github. If you have suggestions for improvement, let us know by opening issues there. And if you like Literapy we would be happy if leave us a star on Github.

Why did we build Literapy

We believe that books can help to overcome difficult situations. From the sense of comfort knowing that you’re not the only going through a difficult situation to the calming effect of prose — reading helps. ( Bibliotherapy is in fact a well established concept)

However, finding the right book for your situation can be a tedious task. This is, why we decided to build Literapy. We wanted to provide you with a tool that helps you to find the book you need with the click of a button!

We hope you enjoy exploring books with Literapy.
